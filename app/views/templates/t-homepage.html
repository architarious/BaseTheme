Objective structures

In structuation CSS, objective structures are the laws, algorithms, base elements and architectures that all web components are built from.

Base

The base level handles the importation of vendor function and mixin libraries. It's also where other frameworks like bootstrap would be loaded.

Styleguide

Branding, html element styling and variable declarations all happen on the styleguide level

Objects

Objects are fundamental UX component architectures that aren't tailored to a certain kind of content.
