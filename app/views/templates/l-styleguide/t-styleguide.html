Grids

This theme uses Singularity for rapid development of responsive grids that use CSS Grid, but still work in IE and other browsers that don't yet support css-grid.

A small number of grid utility classes have also been created to assist with theme development when singularity mixins can't be formally coded. All utility grid classes snap back to single column on viewports below 640px.
