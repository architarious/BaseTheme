Slick-Slider is an accessible and responsive slider developed by Ken Wheeler. It has multiple different variations and is very easy to both implement and customize. For more details about slick slider, go to https://kenwheeler.github.io/slick/

Auto-slider is a simple CSS image slider that rotates images. It's mixin allows you to customize the delay and number of slides that it switches. It isn't a part of slick-slider.
